* Profile Evidence

Ниже только то, что подтверждается кодом и профилем.

** Hot path 1: notification -> fragment update

Профиль показывает цепочку:

`timer-event-handler -> apply -> acp--route-incoming-message -> agent-shell--on-notification -> agent-shell--update-fragment -> agent-shell-ui-update-fragment`

Это значит, что latency в основном приходит от обработки входящих событий ACP, а не от пользовательских команд.

** Hot path 2: fragment lookup

`agent-shell-ui-update-fragment` ищет существующий фрагмент через `text-property-search-backward`, а потом `agent-shell-ui--read-fragment-at-point` повторно вычисляет блок через `agent-shell-ui--block-range`.

Фактический код:

`agent-shell-ui.el:84-90`
`agent-shell-ui.el:96-111`
`agent-shell-ui.el:212-231`

** Hot path 3: block range discovery

`agent-shell-ui--block-range` не хранит готовый диапазон, а заново ищет его по `agent-shell-ui-state`.

Фактический код:

`agent-shell-ui.el:219-231`

** Hot path 4: range search helper

`agent-shell-ui--nearest-range-matching-property` делает backward search, а потом forward search.

Фактический код:

`agent-shell-ui.el:233-251`

** Hot path 5: header rebuild

Профиль фиксирует `agent-shell--update-header-and-mode-line` на 10%, а внутри есть пересборка header-model и mode-line menu.

Фактический код:

`agent-shell.el:3445-3458`
`agent-shell.el:3160-3435`
`agent-shell.el:6167-6226`

** Hot path 6: markdown overlays

`agent-shell--update-fragment` всегда запускает `markdown-overlays-put` для body/right label.

Фактический код:

`agent-shell.el:2879-2970`

** Hot path 7: GC

`Automatic GC` в профиле подтверждает, что горячий путь много аллоцирует.

Вероятные источники аллокаций в коде:

`map-elt`
`map-nested-elt`
`format`
`concat`
`buffer-substring`
`split-string`

** What config can change now

Подтвержденные переключатели, которые уменьшают часть стоимости:

- `agent-shell-header-style`
- `agent-shell-show-busy-indicator`
- `agent-shell-show-context-usage-indicator`
- `agent-shell-show-session-id`
- `agent-shell-show-config-icons`
- `agent-shell-prefer-viewport-interaction`
- `agent-shell-section-functions`
- `agent-shell-highlight-blocks`

Важно: ни один из них не убирает сам повторный поиск фрагментов и блоков.
